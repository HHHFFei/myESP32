# myESP32
这是我的ESP32实验基地。为了方便，同时也可能是由于对仓库的使用还不够了解，所有不同的项目都存放在这一个仓库下
## 1. 模块
涉及到的模块包括：  
1. ESP32-CAM  
2. ESP32-C3mini
3. LCD（LCD液晶屏模块）  
4. MPU6050（六轴陀螺仪模块）  
5. MLX90640（热成像镜头模块）  
6. INM441（全向麦克风模块）  
7. WS2812B（LED灯珠）
